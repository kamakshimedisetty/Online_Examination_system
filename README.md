Online MCQ Examination system -python

Description:
This project is a console-based examination system developed using python. It allows a student to log in, attempt MCQ questions,and get instant evaluation with a score card.

Features:
- Student login
- Randomized MCQ questions
- Auto evaluation
- Pass/Fail result
- Scorecard generation

Technologies Used:
- Python 3
- Lists, Dictionaries, Loops, Functions

Learning Outcome:
This project helped me understand basic Python programming,
control structures, and problem-solving skills.
